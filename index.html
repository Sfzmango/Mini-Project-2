CSS Media Query

This is used to check for specific properties from a media device's UI.
@media screen and (max-width: 745px) { body { --bFont: rgb(126, 103, 103); --bkgd: rgb(253, 243, 196); --boxbrdr: olive; } }

CSS Hover Glow

This is used to make boxes glow when hovering on this page.
.css-snip-flex section:hover, .css-snip-flex section:hover h2 { box-shadow: var(--shadow) maroon; transition: box-shadow 350ms; } }

CSS Universal Selector

This is used to give every element within the page this style.
* { font-family: monospace; }

CSS Root Variables

This is used to create variables for ease of use for repeating styles on this page.
:root { --bFont: #217e7e; --mrgn: 15px; --shadow: 0px 0px 20px 3px; --thinb: 1px solid sienna; }

CSS Flexbox

This is used to create the flexbox on this page.
.css-snip-flex { display: flex; justify-content: space-between; flex-wrap: wrap; padding: 25px; }

CSS Linear-Gradient

This is used to create the linear-gradient effect on this page.
code { background: linear-gradient(var(--bkgd), var(--bFont)); }
